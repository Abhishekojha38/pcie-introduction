# PCIe Introduction & Troubleshooting

This repository documents practical lessons, guides, and troubleshooting notes for PCIe (Peripheral Component Interconnect Express) development, focusing on software fundamentals and real-world debugging.

## Contents

### [Getting Started with PCIe](PCIe-Introduction.adoc)
A practical beginner's guide covering:
- **Core Concepts**: Difference between legacy PCI and PCIe, packet-based communication, and layered architecture.
- **Components**: Root Ports, Switches, Endpoints, and Bridges.
- **Topology analysis**: Understanding PCIe hierarchies.
- **Hands-on with QEMU**: Instructions for exploring PCIe hierarchies using QEMU on ARM64.
